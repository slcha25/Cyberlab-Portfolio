QUESTIONS:
1. What Nmap flag did you use to find the specific software versions running on these ports?
Answer: -sV (version detection) 

2. Why is it dangerous to run an intense scan (like -A) on a fragile legacy system?
Answer: It is dangerous to run an intense scan on a fragile legacy system because -A is enable OS detection, 
        version detection,script scanning and traceout. These probes send unusual, malformed or out-of-order packets
        that legacy systems(old routers, medical devics, SCADA/ICS equipment) were never designed to handle.
	Thsi can cause crashes, reboots, or denial of service(DoS).In addition, some scripts may trigger 
	authentication locakouts or log flooding. Old systems with limited disk space can have their entire log
	partittion filled by an aggressive Nmap scan. This may cause the system to halt operation or fail to record
	real security events.
