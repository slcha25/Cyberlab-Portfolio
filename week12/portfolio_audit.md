## 💭 Professional Reflection

This portfolio documents eleven weeks of intensive, hands-on cybersecurity training spanning the full attack-and-defense spectrum from web application exploitation and CVE-based vulnerability analysis to enterprise-grade active defense using industry tools. The breadth of work demonstrated across offensive disciplines (SQL injection, XSS, CSRF, API security, and Metasploit exploitation frameworks) and defensive operations (Suricata IDS/IPS, Sysmon endpoint monitoring, SIEM/KQL log analysis, and memory and disk forensics) reflects a practitioner who can reason across both sides of a security engagement. A particular source of pride is the Week 11 Suricata lab, in which a container permission failure silently blocking custom detection rules was independently diagnosed and resolved through systematic Linux troubleshooting, an experience that deepened both technical skill and practitioner confidence. Going into Phase 2, the primary areas identified for continued growth include strengthening Python scripting proficiency to automate security workflows and developing more rigorous forensic documentation habits to meet professional incident reporting standards.
