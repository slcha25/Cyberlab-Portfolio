## Why Lists Beat Variables at Scale

List is a tool to keep all variables in a place. If we have 100 separater variable,
we need to name and input everytime. However, list can help us organize all variable in a place.
When you input the update vairable into the list, the rest of variable still in the list and no 
need to input it again. In addition, with a list, you write the code once and tell python to run 
every item in once time. You don't need to check 100 separate vairables 100 times. It save time. 
The advantage of List are organazied easily, automate efficiency and scalability.
