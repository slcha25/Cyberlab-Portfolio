## What I Learned

- In S1 Lab, I learned how to navigate a headless Linux filesystem using absolute and relative paths without relying on a Graphical User Interface (GUI),Understand and explain the Filesystem Hierarchy Standard (FHS), identifying the purpose of path,and execute core terminal commands

- In S2 Lab, I learned how to decipher symbolic permission strings and translate them into their numerical Octal equivalents,manipulate file and directory access controls using the chmod command, and safely execute administrative commands using sudo to bypass standard user restrictions
  
- In S3 lab, I learned how to manipulate standard input (stdin), output (stdout), and error (stderr) using redirection (>, >>) and piping (|), extract specific forensic data from massive text files using grep and regular expressions, and automate the formatting of data streams using awk to isolate actionable intelligence.
